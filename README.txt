پروژه: سیستم مدیریت نود با درخت و صف اولویت (نسخه دانشجویی)

فایل‌ها:
- main.go: کد اصلی سمت سرور
- index.html: رابط گرافیکی قابل اجرا در مرورگر
- nodes.db: دیتابیس SQLite (خودکار ساخته می‌شود)

نحوه اجرا:
1. نصب Go
2. اجرای برنامه با: go run main.go
3. باز کردن index.html در مرورگر

در صورت اجرای روی سرور، در index.html مقدار API را به IP سرور تغییر دهید.
